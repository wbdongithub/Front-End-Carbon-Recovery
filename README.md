# Front End Carbon Recovery — Version 1.0

This repository is configured for GitHub Pages deployment using GitHub Actions.

## Local preview

```bash
npm install
npm start
```

## Production build

```bash
npm run build
```

## GitHub Pages

1. In the repository, open **Settings → Pages**.
2. Set **Source** to **GitHub Actions**.
3. Push the project files to the `main` branch.
4. The included workflow builds and deploys the site automatically.

## Updating with GitHub Desktop

Replace the existing project files in the cloned repository folder, but preserve the hidden `.git` folder. Then commit and push the changes from GitHub Desktop.
