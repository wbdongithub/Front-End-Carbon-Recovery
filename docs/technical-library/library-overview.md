# Technical Library Overview

The FECR website should remain readable for non-specialists, but it should also be supported by technical references.

The library should be organized by topic rather than by document type.

## Suggested sections

- Microscreens / Rotating Belt Filters
- Cellulose in wastewater
- Aeration and downstream process effects
- Dewatering
- Heat-pump drying
- Pyrolysis
- Emissions and off-gas treatment
- Economics
- Demonstration results
- Case studies

## Editorial rule

Important claims should be supported with data, references, photos, diagrams, or case studies whenever possible.

When evidence is still developing, the site should present the point as an engineering expectation or hypothesis rather than as a proven result.
