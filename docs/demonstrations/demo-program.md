# Demonstration Program

The FECR concept benefits from physical demonstrations because the microscreen process is easy to understand once people can see it operate.

## Transparent model

The see-through microscreen model shows how fibers are captured on a moving belt while smaller particles pass through. It is useful for explaining the basic operating principle to operators, engineers, and decision makers.

## Portable demonstration trailer

A trailer-mounted demonstration unit allows testing at real wastewater treatment plants. This is important because a microscreen may be installed at more than one possible location in the treatment process.

The demonstration unit can help evaluate:

- Raw wastewater after grit removal
- Primary solids streams
- Other solids-rich sidestreams
- Dewatering performance
- Solids capture
- Operator concerns
- Practical layout issues

## Why demonstration matters

Utilities often need to see site-specific data before supporting a new process. A portable unit can turn FECR from an abstract concept into a measurable plant-specific option.
