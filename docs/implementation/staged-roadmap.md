# Staged Implementation Roadmap

FECR does not have to be built all at once.

This is one of its most important advantages. A utility can begin with the stage that solves its immediate problem, then add other stages later as funding, regulations, and operating experience justify them.

## Stage 1: Install a microscreen

Goal: relieve solids loading.

This stage can reduce the solids load on downstream treatment processes and produce a dewaterable primary solids stream.

## Stage 2: Improve solids management

Goal: manage the recovered solids better than conventional back-end sludge.

Depending on local regulations and testing, interim management may include landfill disposal, beneficial reuse, or integration into an existing biosolids program.

## Stage 3: Add heat-pump drying

Goal: make stable, transportable dried solids.

Drying can reduce hauling costs, improve storage, and produce a feedstock suitable for regional processing or future pyrolysis.

## Stage 4: Add pyrolysis

Goal: destroy contaminants, reduce volume, recover energy, and produce biochar.

Pyrolysis can be added when the utility is ready to move from improved solids handling to more complete carbon recovery.

## Stage 5: Develop biochar uses

Goal: create value from the carbon product.

This stage belongs mostly in the future Biochar Knowledge Base, but the FECR site should identify it as a long-term opportunity.
